# fukui-kanko-trend-data

福井県内の観光関連データを提供するリポジトリです。観光地の検索アクセス状況や、利用者の行動分析データを確認できます。

## データ・API
このリポジトリには以下のデータファイルが含まれています:

- `2024/metadata.json`: 2024年の福井県内の観光地情報
- `2025/metadata.json`: 2025年の福井県内の観光地情報
- `2026/metadata.json`: 2026年の福井県内の観光地情報
- `2025/area_永平寺町_d67f29ed-a1eb-4178-8503-2800e3f8497b_daily_metrics.csv`: 2025年の永平寺町の観光データ
- `2024/area_永平寺町_d67f29ed-a1eb-4178-8503-2800e3f8497b_daily_metrics.csv`: 2024年の永平寺町の観光データ
- `2026/area_永平寺町_d67f29ed-a1eb-4178-8503-2800e3f8497b_daily_metrics.csv`: 2026年の永平寺町の観光データ

これらのデータには、観光地の検索、アクセス状況、レビューなどの情報が含まれています。
