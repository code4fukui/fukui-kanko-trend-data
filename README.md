# fukui-kanko-trend-data

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

This repository contains tourism-related data for various areas in Fukui Prefecture, Japan. The data includes daily metrics such as map views, search views, directions, call clicks, website clicks, average ratings, and review count changes.

## Data / API
The data is organized into yearly directories (`2024`, `2025`, `2026`) and each directory contains a `metadata.json` file that lists the available areas and their corresponding data files. The area data files are named in the format `area_{area_name}_{area_id}_daily_metrics.csv` and contain the daily metrics for each area.

## Usage
To use the data, you can access the CSV files directly from the repository. The `metadata.json` files provide information about the available areas and their corresponding data files.
